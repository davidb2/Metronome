Metronome
=========

This program is an acting metronome.

Usage
-----
```
    Usage:
        metronome <beats_per_minute> <beats_per_measure>
```

Installation
------------
  1. Run:
```
fsc metronome.fs
```
  2. Then (for example), run:
```
metronome 120 4
```
The output should be:
```
TICK tick tick tick
TICK tick tick tick
TICK tick^C
```
